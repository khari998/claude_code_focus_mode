Complete Setup

Run this command in your terminal to install the extension in Claude Code:

curl -fsSL https://raw.githubusercontent.com/khari998/claude_code_focus/main/scripts/install.js | node

Want to see what this does? View the source code on GitHub

1 Copy and run the command above
2 Restart any active Claude Code instances to use the extension
3 Start using Claude Code - sites will unblock automatically

You're all set!

The daemon is running and connected.

Don't forget to restart any active Claude Code instances to activate the extension.
